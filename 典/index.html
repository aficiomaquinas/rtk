code

Reading:

On-Yomi: テン、デン

Heisig story:

We introduce this character here because of its connection to the book-related kanji treated above. It is based on the character for bend (frame 1256), whose last stroke is lengthened to coincide with the first stroke of the element for tool.

Koohii stories:

1) [rgravina] 15-10-2006(178): Coding is a tool which you can use to bend the computer to your will.

2) [vileru] 8-1-2010(75): This kanji means a code of law, not a secret language or symbolic language (such as programming code) … Story: The code of law is a tool used to bend citizens to society's will. You can use the image of a totalitarian state, but be careful not to confuse it with the pinnacle primitive Heights (#1294 阪).

3) [fuaburisu] 8-7-2008(60): Think "codex" (mnemonic help) or "code of laws", or even Leonardo Da Vinci's code. Here I see a codex as a tool used to bend the masses into submission. Picture a huge codex book lifted down on crowds to bend them down in submission. Another: if you bend to the code without asking questions, you are a tool!

4) [jotun] 8-7-2008(23): Bender can be used as a tool for bending. But only if you know the right code (He never does want you want to).

5) [tristanthorn] 18-3-2009(20): Code/rules can be bent with the correct tool… (usually money).
